# Codex Composition Outer Probe

Invoke `$axis-codex-composition-inner`. Write the exact token it returns to
`codex-composition-artifact.txt` in the current workspace. Then reply with
exactly `AXIS_CODEX_COMPOSITION_OUTER_OK_D7N5` and no other text.
